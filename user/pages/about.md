## About Me

Hi! I'm **DM**, a developer passionate about web technologies, game design, and clean architecture.

### What I Do

- Build fast, accessible static websites with Astro
- Write about development workflows and coding best practices
- Experiment with pixel art and indie game design

### This Blog

This blog is built with a custom architecture that separates personal data (`user/`) from application code (`src/`). This makes the content fully portable — if I ever switch to a different static site generator, I just take the `user/` folder with me.

### Tech Stack

- **Astro 7** — Static site generator
- **Tailwind CSS v4** — Utility-first styling
- **TypeScript** — Type safety
- **Markdown** — Content authoring

### Get In Touch

Feel free to reach out via GitHub or Twitter. I'm always happy to chat about web dev!
