## 为什么选择 Astro？

Astro 是一个现代的静态站点生成器，默认不发送任何 JavaScript。非常适合内容优先的网站。

### 核心特性

- **岛屿架构**：只水合需要交互的部分
- **默认 SSG**：纯静态 HTML 输出，性能最大化
- **框架无关**：可以使用 React、Vue、Svelte 或纯 Astro 组件

## 为什么选择 Tailwind CSS？

Tailwind CSS v4 带来了 Vite 原生插件方案，更快更干净。不再需要 PostCSS 配置文件。

## Why Astro?

Astro is a modern static site generator that ships zero JavaScript by default. It's perfect for content-focused websites like blogs.

### Key Features

- **Island Architecture**: Only hydrate the interactive parts of your page
- **SSG by Default**: Pure static HTML output for maximum performance
- **Framework Agnostic**: Use React, Vue, Svelte, or plain Astro components

## Why Tailwind CSS?

Tailwind CSS v4 brings a Vite-native plugin approach that's faster and cleaner. No more PostCSS configuration files.
